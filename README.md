# Q4-1005-Gustpill
Aquest projecte consisteix a treballar amb un dataset de la UAB, analitzant la informació amb Python per explorar el campus i preparar dades per a la creació d’un joc. L’objectiu del nivell 1 és comprendre les dades, identificar patrons i zones rellevants, i obtenir estadístiques bàsiques.

Les tasques realitzades inclouen identificar zones hotspot amb major densitat, analitzar patrons temporals com hores punta i dies amb més activitat, visualitzar la distribució per edifici i generar estadístiques bàsiques del campus.
